Dies ist die Übersichtsseite für Autoren

Die Daten dieser fiktiven Autoren sind eine Collection, die in der `_config.yml` als solche angegeben wurde. Die Dateien der Autoren befinden sich im Verzeichnis `_authors` und haben als Template `authors.html`. Durch diese Vorgehensweise wird für jeden einzelnen Autor automatisch eine eigene Seite generiert. Gibt es einen `Nickname`, wird der für den Titel und in der Navigation benutzt.

{% assign authors = site.authors %} {% for author in authors %} {% if author.nickname %}

{{ author.nickname }}

{% else %}

{{ author.title }}

{% endif %}

{{ author.description }}

Zum Autor{% if author.nickname %} {{ author.nickname }}{% endif %}{% endfor %}
